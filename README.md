# React + TypeScript + Vite + WebSpatial

This template provides a minimal setup to get React working in Vite with WebSpatial SDK.

## Common Commands

- `pnpm dev`
- `pnpm dev:webspatial`: only for packaging website into a packaged hybird App with built-in WebSpatial Runtime on visionOS emulator
- `pnpm build`
- `pnpm lint`
